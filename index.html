TMC-CLI New

Test My Code Command Line Interface

TMC-CLI is a tool for using TestMyCode in command line. Current version is only for Linux & Mac OSX

Installation instructions

curl https://raw.githubusercontent.com/rage/tmc-cli/master/scripts/install.sh | bash

Now you're ready to type commands.

How to get started with a course

  • Set server address with tmc set server <serveraddress>.
  • Login with tmc login.
  • List available courses with tmc list courses. Now you get explicit name of course.
  • Download course with tmc download <coursename>.
  • Set current course tmc set course <coursename>.
  • Now you're ready to write code. Test your code in project folder with tmc test.
  • Submit your code with tmc submit.

Usage

Login
List courses to get ID of course

Download exercises by course ID

Run Tests in project folder
Submit in project folder

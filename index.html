TMC-CLI New

Test My Code Command Line Interface

TMC-CLI is a tool for using Test My Code in command line. Current version is only for Linux & Mac OSX

Installation instructions

curl https://raw.githubusercontent.com/rage/tmc-cli/master/scripts/install.sh | bash

Now you're ready to type commands.

How to get started with a course

  1. Set server address with 'tmc set server '.
  2. List available courses with 'tmc list courses'. Now you get explicit name of your course
  3. Download course with 'tmc download '
  4. Set course to yout current course with 'tmc set course '
  5. Now you're ready to code. Test your code in folder with 'tmc test'
  6. Submit your code with 'tmc submit'

Usage

Login
List courses to get ID of course

Download exercises by course ID

Run Tests in project folder
Submit in project folder
